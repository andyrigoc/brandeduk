# Customization Position Images

This folder contains position images for each product category.

## Folder Structure

```
positions/
├── adult-tops/          # T-Shirts, Polos, Hoodies, etc.
├── youth-tops/          # Youth T-Shirts
├── infants/             # Baby clothing
├── pants/               # Work Trousers, Shorts
├── headwear/            # Caps, Beanies
├── bags/                # Tote bags, Backpacks
├── towels/              # Beach towels, Hand towels
├── blankets/            # Blankets
├── mouse-mats/          # Mouse mats
├── aprons/              # Aprons
└── non-apparel/         # Awards, Coasters, etc.
```

## Image Naming Convention

Each position image should be named using kebab-case:
- `body.png`
- `back.png`
- `left-sleeve-short.png`
- `right-sleeve-long.png`
- `left-chest.png`
- etc.
